## [1.0.0] - 2025-12-19

### Added
- Initial release
- OAuth 2.0 client credentials authentication
- `piwik_list_sites` - List all tracked sites/apps
- `piwik_analytics_summary` - Get sessions, pageviews, visitors, bounce rate
- `piwik_top_pages` - Get most visited pages
- `piwik_traffic_sources` - Get traffic source breakdown
- `piwik_goals` - Get goal conversions and revenue
- `piwik_custom_query` - Flexible custom analytics queries
- `piwik_available_columns` - Reference list of common dimensions/metrics
